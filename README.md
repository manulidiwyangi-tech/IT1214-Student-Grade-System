# Student Grade Management System Assignment

This is my Java project for the IT1214 assignment. It is a console application to manage student marks and grades using arraylist.

## What this program can do:
1. Add a new student with ID, Name, and Marks.
2. View all student details that already entered.
3. Search student details by entering their Student ID.
4. Calculate total marks and the average marks of students.

## How to run this code:
Open your cmd or terminal in this folder and type:
javac Main.java
java Main
